Installing a CPU

1. Locate the golden triangle on you're cpu. this will need to match with the golden triangle on you're processor.
2. Open the socket cover.
3. Place the cpu in the socket with the correct alignment, this should require no force at all.
4. Push down the retention arm all the way and lock it. this will require force in locking this down, this is perfectly normal as long as it is seated properly.
5. You're now done installing a CPU!
